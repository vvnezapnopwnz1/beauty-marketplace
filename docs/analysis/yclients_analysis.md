## 🗂️ Общая структура платформы

## 📋 Детали каждого раздела

### 1. Журнал записи (`/timetable`)

- **Вид:** Временная сетка — один столбец = один сотрудник
- **Режимы отображения:** День / Неделя
- **Фильтр:** По должностям
- **Верхняя панель:** кнопки "Продать", касса (0 ₽), поиск, настройки отображения
- **Записи:** Цветные блоки с названием услуги и временем (пример: "Стрижка 10:00–10:30")
- **Метаданные записи:** Клиент, телефон, статус ("Ожидание клиента"), источник ("Web-интерфейс")
- **Сотрудники:** у каждого своя колонка с должностью (Лера — Косметолог, Петров Андрей — Парикмахер)
- **Дополнительно:** кнопка "Настроить график работы", кнопка "Добавить сотрудников в день"

---

### 2. График работы (`/work_schedule`)

- **Вид:** Сетка-календарь (Месяц / Неделя)
- **Строки:** По сотрудникам
- **Ячейки:** Рабочее время зелёным цветом (10:00–22:00, 10:00–18:00 и т.д.)
- **Счётчики:** Количество смен и суммарные часы (например: 108 ч., 140 ч.)
- **Фильтры:** По сотрудникам / должностям
- **Выгрузка:** PDF-экспорт
- **Мини-цифры в ячейках:** Количество записей на конкретный день
- **Навигация:** стрелки вперёд/назад, кнопка "Сегодня"

#### Фильтры:

#### Поиск:

- Скидка (%)

- Дата рождения (день / месяц / год)

### 4. Аналитика (`/analytics/{id}`)

#### Основные метрики (карточки):

- **Средняя заполненность** — средняя заполненность рабочего дня

#### Фильтры:

- Все должности
- Все сотрудники
- Все сотрудники с доступом

## 💡 Интересные UX-решения для заимствования

1. **Быстрая панель** — 4 кнопки "продать/платёж/услуги/товары" всегда видны в боковом меню без перехода на другую страницу
2. **Мини-календарь** в боковом меню — при наведении на иконку раздела открывается мини-вью
3. **Бейджи источников** в журнале записей — яркий тег "WEB-ИНТЕРФЕЙС" / "ОНЛАЙН" у каждой записи
4. **Сегменты клиентов** одной строкой над таблицей — Новые / Повторные / Потерянные / Заканчивается абонемент
5. **Лента активности** на дашборде с типом события (Создание / Изменение) и деталями diff
6. **Важность клиента** с системой металлов (Бронза / Серебро / Золото)
7. **Счётчики часов** в графике работы — сразу видно сколько часов у каждого сотрудника за месяц
8. **Переключатель Быстрая панель** (toggle) в нижней части меню — можно включить/выключить
