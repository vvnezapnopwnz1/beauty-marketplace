# Конкурентный анализ: Booksy vs Яндекс.Карты vs Наша платформа

> Обновлено: апрель 2025  
> Контекст: разработка beauty-маркетплейса для России / СНГ, старт с Москвы

---

## 1. Что умеет Booksy — полная карта фич

### Для клиента
| Фича | Описание |
|---|---|
| Геопоиск + фильтры | Услуга, цена, рейтинг, дальность |
| Профиль мастера | Портфолио работ, отзывы, прайс, доступное время |
| Онлайн-запись 24/7 | С выбором конкретного мастера, даты, времени |
| Листист ожидания | Клиент встаёт в очередь, при отмене — автоуведомление |
| Booking из соцсетей | Кнопка «Забронировать» прямо в Google, Instagram, Facebook |
| История записей | Перезапись в один тап |
| Семейные записи | Можно записать несколько человек |

### Для мастера / салона
| Фича | Описание |
|---|---|
| Депозит при записи | Клиент платит часть суммы при бронировании |
| Штраф за неявку | Автоматическое списание по карте-на-файле |
| No-show статистика | По данным Booksy: −20% отмен после включения |
| CRM | Карточка клиента, история, заметки, теги, фото |
| Блокировка клиентов | «Проблемные» не могут записаться онлайн |
| SMS / Email рассылки | По своей базе клиентов |
| Аналитика | 16 типов отчётов: доход, загрузка, прогноз выручки |
| Буст в выдаче | Pay-per-result: 30% только с новых клиентов, не подписка |
| Интеграция с Google/Apple Calendar | Нет двойных записей |

### Чего нет у Booksy для России
- Ушли с рынка после санкций 2022 года
- Нет интеграции с рос. платёжками (СБП, ЮКасса, Тинькофф)
- Нет Telegram-уведомлений
- Нет адаптации под самозанятых мастеров (налоговый режим РФ)

## 2. Что умеет Яндекс.Карты — и чего катастрофически нет

### Есть
- Геопоиск организаций (название, адрес, координаты)
- Контакты, часы работы
- Фото организации (общие, не портфолио работ)
- Отзывы и средний рейтинг
- Маршрут до салона

### Нет — критические пробелы
| Чего нет | Почему это важно |
|---|---|
| Онлайн-записи | Клиент вынужден звонить — высокий отток |
| Фильтра по услуге | Нельзя найти «маникюр гель-лак до 2000₽ рядом» |
| Профиля мастера | Клиенты ходят к людям, не к юрлицам |
| Прайса и списка услуг | Непонятно сколько стоит и что предлагают |
| Портфолио работ | Нет визуального доказательства качества |
| Верификации отзывов | Отзывы накручивают — известная боль рынка |
| Напоминаний о записи | 40–50% no-show в нишах без напоминаний |
| Инструментов для бизнеса | Яндекс — справочник, не платформа |

**Вывод:** Яндекс.Карты решают задачу «найти адрес», а не «записаться к нужному мастеру». Это телефонный справочник с геолокацией.

## 3. Наши уникальные фичи — чего нет нигде на рынке СНГ

### 🏆 Фича 1: Профиль мастера ≠ профиль салона

**Проблема, которую решает:**  
В реальности клиенты ходят к конкретной Насте или Диме, а не в «Студию красоты №5». Если Настя уходит — клиент уходит вместе с ней. Яндекс не знает о существовании Насти. YCLIENTS знает, но не показывает клиентам.

**Что мы делаем:**  
Мастер получает свой публичный профиль с накопленными отзывами, портфолио и рейтингом. Он привязан к салону, но при смене работы его репутация остаётся с ним — клиенты находят его на новом месте через ту же платформу.

**Почему это важно для роста платформы:**  
Мастер сам заинтересован в развитии своего профиля и приводит своих клиентов на платформу.

---

### ⚡ Фича 2: «Свободно прямо сейчас»

**Проблема, которую решает:**  
Пятница, 18:00, хочу подстричься сегодня вечером. В Яндексе ты не знаешь, кто свободен. У конкурентов нет маркетплейса с расписанием. Спонтанный спрос никто не обслуживает.

**Что мы делаем:**  
Фильтр: «Есть слоты сегодня / завтра / на этой неделе». Один тап — список мастеров, которые реально доступны прямо сейчас.

**Двойная ценность:**  
- Клиенту: моментальное решение спонтанного спроса
- Мастеру: заполнение пустых окон без обзвона базы

---

### ✅ Фича 3: Верифицированные отзывы

**Проблема, которую решает:**  
Накрутка отзывов в Яндексе и 2ГИС — публично известная и массовая практика. Клиенты не доверяют рейтингам. Хорошие мастера страдают от недобросовестной конкуренции.

**Что мы делаем:**  
Отзыв можно оставить только после подтверждённой записи через платформу. Оценка «5.0 на основе 200 верифицированных визитов» vs «4.8 из отзывов» — это разные вещи.

**Почему это конкурентное преимущество:**  
Честный рейтинг — это аргумент, с которым качественные мастера сами будут продвигать платформу.

---

### 📱 Фича 4: Telegram вместо SMS

**Проблема, которую решает:**  
SMS дорого (3–7₽ за сообщение), открываемость падает, номера блокируются. Email открывают единицы. В России у 80%+ аудитории установлен Telegram.

**Что мы делаем:**  
Напоминания о записи, подтверждения, уведомления о свободных слотах — через Telegram Bot. Для мастеров: уведомление о новой записи тоже в Telegram.

**Экономика:**  
- SMS: ~5₽/сообщение × 3 уведомления = 15₽ на визит
- Telegram: 0₽
- При 100k записей/мес экономия ~1.5М₽ в год

---

### 🧑‍💼 Фича 5: Самозанятые мастера как отдельный тип профиля

**Проблема, которую решает:**  
Огромный сегмент рынка — мастера без салона: работают на дому, в аренду кресла, на выезде. YCLIENTS и DIKIDI ориентированы на организации. У самозанятых нет юрлица, нет ресепшена. Им нужен простой личный профиль.

**Что мы делаем:**  
Тип аккаунта «Мастер» — без привязки к салону, с указанием «работаю на дому / в аренду / выезд». Онлайн-запись, портфолио, чек самозанятого через ЮКасса.

## 4. Карта фич: MVP vs Следующий этап

### ✅ В MVP
- [ ] Каталог: профиль салона + профиль мастера
- [ ] Геопоиск + фильтры (услуга, цена, дальность, рейтинг)
- [ ] Фильтр «Свободно сегодня / завтра»
- [ ] Онлайн-запись (только для платных профилей)
- [ ] Верифицированные отзывы
- [ ] Напоминания через Telegram Bot
- [ ] Профиль самозанятого мастера
- [ ] Базовый бесплатный профиль (из Яндекс API / 2ГИС)

### 🔜 После валидации
- [ ] Депозит при записи (нужна платёжка: СБП / ЮКасса)
- [ ] Штраф за неявку
- [ ] Листист ожидания с автоуведомлением
- [ ] CRM для мастера (карточка клиента, история)
- [ ] SMS-рассылки по базе (для мастеров без Telegram у клиента)
- [ ] Буст в выдаче (платное продвижение)
- [ ] Аналитика для салона

### ❌ Не копировать у Booksy (overengineering для старта)
- Интеграция с Instagram / Facebook booking (не та аудитория)
- 16 аналитических отчётов
- POS-терминал (оплата в салоне)
- Intake-формы и вейверы
- Групповые бронирования

## 6. Стратегия старта: решение проблемы курицы и яйца

**Шаг 1 — Заполнить каталог до первого мастера:**  
Парсинг / реалтайм запросы к 2ГИС API (мягче лицензия, чем Яндекс). Базовые профили салонов появляются автоматически. Клиент видит каталог с первого дня.

**Шаг 2 — Привлечь первых мастеров в Москве:**  
Таргет: самозанятые мастера маникюра и парикмахеры с аудиторией в Instagram/Telegram. Оффер: «Бесплатный профиль навсегда + онлайн-запись бесплатно первые 3 месяца». Цель — 50–100 активных мастеров в одном районе.

**Шаг 3 — Первые клиентские записи:**  
Когда есть реальные мастера с расписанием → запускаем трафик. Instagram/ВКонтакте таргет по Москве: «Запишись онлайн, не звони».
