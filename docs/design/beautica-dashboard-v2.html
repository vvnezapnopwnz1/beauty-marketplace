Заметки по UI — что нового в v2

Что доработано и ключевые решения по дизайну

Новые разделы

👤 Мастера — список
Карточки мастеров с: аватар + цвет + онлайн-статус, специализация, список услуг, сжатое расписание (пн-вс), bar загрузки (green/yellow/red), 3 ключевые метрики в footer: рейтинг, визиты, выручка/мес. Клик на карточку → детальная страница мастера.
👤 Мастер — детальная страница
Header с аватаром, именем, ролью, 4 большими KPI-метриками. Блок контактов. Блок специализации. Визуальный недельный календарь. Список ближайших записей. Кнопки: Редактировать, Расписание, Аналитика, Деактивировать.
🕐 Расписание — общее салона
Таб-навигация: «Салон (общее)» + индивидуальный таб для каждого мастера. Общее расписание: toggle вкл/выкл для каждого дня, время-пикер начала/конца + обеденный перерыв. Праздники. Длительность слота (15/30/60 мин).
🕐 Расписание — мастер
Каждый мастер имеет свой таб в разделе Расписание. Значок «≠ салон» если время мастера отличается от базового. Кнопка «Сбросить к базовому». Блок «Отпуск/Больничный/Особые дни». Мини-обзор недели в виде цветных блоков.
👤 Модальное окно мастера
Форма создания/редактирования: выбор цвета карточки (7 цветов — отображается в календаре), ФИО + уровень + описание, контакты (телефон + Telegram + email), мультиселект услуг, инлайн-расписание по дням, настройки видимости и Telegram-уведомлений.
📋 Записи — доработан
Табличный вид с фильтрами (Сегодня/Завтра/Неделя/Все), поиск, кнопки «Подтвердить» / «Отмена» у каждой записи прямо в строке. Статус-бейджи, отменённые записи с кнопкой «Восстановить».

Ключевые UX-паттерны

🎨 Цвет мастера в календаре — при создании мастера выбирается цвет. Все его записи в календаре отображаются этим цветом. Как в Google Calendar. Это критично для быстрой визуальной ориентации.

⚡ Статус онлайн/офлайн — зелёная точка у мастера если он сейчас в рабочие часы. Жёлтая = обед. Серая = выходной.

⚠️ «≠ салон» — желтый лейбл в расписании мастера, если его часы отличаются от базовых часов салона. Помогает быстро заметить расхождения.

📊 Загрузка bar — цвет меняется: green (>70%), orange/accent (50–70%), red (<50%). Тревожный сигнал сразу виден на карточке.

🔒 Доступ к дашборду — мастер может иметь ограниченный доступ только к своим записям. Это важно для multi-staff салонов где мастера не должны видеть финансы друг друга.

↩️ Inline-подтверждение в списке записей — кнопки ✓ / ✗ прямо в строке без открытия модального окна. Как в Fresha — экономит время владельца.
